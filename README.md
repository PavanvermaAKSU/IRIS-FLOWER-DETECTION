# 🌸 Iris Flower Classification Platform

A full-stack Machine Learning web application for Iris Flower Classification using FastAPI, Scikit-Learn, React.js, SQLite, and Recharts.

## Features

- Iris Species Prediction
- Confidence Score
- Flower Image Display
- Interactive Dashboard
- Prediction History
- Species Distribution Chart
- Model Comparison Chart
- SQLite Database Storage
- Responsive UI

## Tech Stack

### Frontend
- React.js
- Axios
- React Router DOM
- Recharts

### Backend
- FastAPI
- Scikit-Learn
- Pandas
- NumPy
- SQLAlchemy
- SQLite
